# ⭐ git clone 방법

```bash
cd 경로설정
git clone url (지정하고 싶은 파일 이름)
```

### 📌 url 설명

clone을 할 Git Repository에 가서 Code에 있는 url을 복사해서 붙여 넣으면 된다.
( url 옆에 아이콘을 누르면 바로 복사가 된다. )
